# 高速矩陣運算（Matrix Multiplication Optimization）

本專案目標是從「一般版矩陣相乘」開始，逐步優化成「高速矩陣運算」版本。  
內容特別設計給 **只有 C 語言基礎一點點的大一新生**，從最基本的程式碼開始，慢慢帶你理解效能的差別。

---

## 🎯 專案目標

1. 了解矩陣乘法的運作方式  
2. 實作最基本的三層迴圈矩陣相乘  
3. 使用時間量測比較不同版本的速度  
4. 練習簡單的優化技巧（不使用太難的語法）

---

## 🧩 專案內容

專案包含以下幾個版本，每個版本的難度循序漸進：

### ✅ **1. basic_version/**
最基本版矩陣相乘：三層 for 迴圈  
適合剛學 C 的人，確保理解什麼是矩陣運算。

### 🚀 **2. improved_cache_version/**
改善記憶體存取方式（仍然是簡單的 C 語法）  
可以帶來 2～4 倍加速。

### ⚡ **3. transpose_trick_version/**
使用「先把 B 矩陣轉置」的技巧  
讓 CPU Cache 命中率大幅提升  
加速效果可達 3～6 倍。

（以上內容完全不需要多執行緒、不需要 SIMD、不需要太難技巧，適合大一新生）
